# 코딩 컨벤션 에이전트 주의사항

코딩 컨벤션 에이전트는 코드 스타일, 주석, 네이밍, 문서 규칙의 일관성을 관리한다. 기능 변경은 하지 않는다.

---

## 필수 확인 문서

- `MAP.md`
- `AGENTS.md`
- `CONVENTIONS.md`
- 대상 영역 문서

---

## 역할

- C# 스타일과 주석 규칙을 점검한다.
- 네이밍 규칙 위반을 찾는다.
- Markdown 문서 줄 수 제한을 확인한다.
- 저장소 규칙 위반을 최소 범위로 수정한다.

---

## 점검 항목

- Allman 스타일 적용 여부
- 모든 제어문에 중괄호 사용 여부
- 공개 타입, 메서드, 매개변수의 한글 XML 주석 여부
- 코드 주석이 한글인지 여부
- 비공개 필드 `_camelCase` 여부
- 공개 API `PascalCase` 여부
- Markdown 문서 200줄 이하 여부
- struct null 비교 여부
- Model 레이어 `UnityEngine` 의존 여부

---

## 주의사항

- 기능 로직을 변경하지 않는다.
- 메서드 호출 순서, 조건식 결과, 실행 흐름을 바꾸지 않는다.
- 넓은 범위의 스타일 정리는 사용자가 요청한 경우에만 수행한다.
- 현재 작업과 무관한 기존 위반은 보고만 하고 임의 수정하지 않는다.
